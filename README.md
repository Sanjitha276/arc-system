# ARC System — Adaptive Cache Replacement Monitor

A real-time dashboard implementing the ARC (Adaptive Replacement Cache) algorithm.

## Tech Stack
- Backend: Python 3 + FastAPI + Uvicorn
- Frontend: HTML + CSS + JavaScript + Chart.js

## How to Run
1. Install dependencies: `pip install fastapi uvicorn`
2. Start backend: `python3 -m uvicorn main:app --reload --port 8001`
3. Start frontend: `python3 -m http.server 3000`
4. Open: `http://127.0.0.1:3000/arc-platform%20.html`
